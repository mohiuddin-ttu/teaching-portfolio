# Md Mohiuddin Teaching Portfolio

This repository hosts a self-contained GitHub Pages teaching portfolio for Md Mohiuddin.

## Live Site Setup

1. Create a new GitHub repository, for example:
   `teaching-portfolio`

2. Upload these files:
   - `index.html`
   - `style.css`
   - `assets/`
   - `materials/`

3. Go to:
   **Settings → Pages**

4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

5. Save. GitHub will generate a public URL.

## Replace Placeholder Files

Replace:

- `assets/Md_Mohiuddin_CV.pdf`

- `materials/monopoly-price-discrimination-notes.pdf`

with your actual information.
