# Md Mohiuddin Teaching Portfolio

This repository hosts a self-contained GitHub Pages teaching portfolio for Md Mohiuddin.

## Live Site Setup

1. I create a new GitHub repository, for example:
   `teaching-portfolio`

2. I upload these files:
   - `index.html`
   - `style.css`
   - `assets/`
   - `materials/`

3. I gGo to:
   **Settings → Pages**

4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

5. I saved it in GitHub, which generates a public URL.
